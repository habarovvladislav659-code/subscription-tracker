# Personal Finance Tracker

WPF приложение для учета личных доходов и расходов.

## Как запустить

1. Скачайте проект и откройте `PersonalFinanceTracker.sln` в Visual Studio 2022
2. Нажмите "Восстановить" когда появится запрос о NuGet пакетах
3. Нажмите F5 или "Запуск" для сборки и запуска

## Что нужно для работы

- Visual Studio 2022
- .NET 6.0 или новее
- Доступ в интернет для загрузки пакетов

## Функциональные возможности

1. **Просмотр транзакций** — список всех транзакций, отсортированных по дате
2. **Добавление транзакции** — указание даты, суммы, категории и описания
3. **Удаление транзакции** — удаление выбранной транзакции
4. **Фильтрация по категории** — фильтрация списка по выбранной категории
5. **Отображение баланса** — общее сальдо (сумма всех транзакций)

## Технологии

- WPF, паттерн MVVM
- SQLite, Entity Framework Core
- xUnit для тестирования
- Moq для моков
- MVVM Toolkit

## Ответы на вопросы
1. Архитектура и тестируемость
Интерфейс ITransactionRepository позволяет легко подменять репозиторий мок-объектом в тестах, изолируя тесты ViewModel от базы данных и соблюдая принцип инверсии зависимостей.

2. Выбор технологий
SQLite + EF Core: Легковесная, не требует установки СУБД, работает "из коробки". Идеально для локального приложения.

3. Моки и изоляция
Mock репозитория позволяет тестировать только логику ViewModel (быстро, независимо). Реальная БД — для интеграционных тестов. Для unit-тестов ViewModel предпочтительны моки.

4. Преимущества MVVM
Разделение UI и логики, упрощение тестирования ViewModel (без UI), привязка данных уменьшает код.

5. Возможные улучшения

-  Экспорт отчетов (PDF/Excel)

-  Графики статистики

-  Резервное копирование

-  Уведомления о расходах

-  Мобильная версия
